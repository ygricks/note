## App
App for own notes in calendar

## Installation

```bash
npm install
```

## Running the app

#### development
```bash
npm run start
```
#### watch mode
```bash
npm run start:dev
```
#### production mode
```bash
npm run start:prod
```

## Test

#### unit tests
```bash
npm run test
```
#### e2e tests
```bash
npm run test:e2e
```
#### test coverage
```bash
npm run test:cov
```
